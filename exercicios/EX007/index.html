Principais formatações

Negrito/Destaque

Desse jeito consigo colocar um texto em negrito utilizando a tag b (NAO SEMANTICA)

Dessa outra forma, utilizando a tag (STRONG) pode se colocar o o termo em negritode forma semantica e correta

Textos em italico

Dessa forma podemos colocar um termo em italico, utilizando a tag (i), FORMA NÃO SEMANTICA

E nessa forma podemos utilizar o termo (em) para colocar um texto em italico, FORMA SEMANTICA

Marcação de texto

Aqui podemos marcar um texto utilizando a tag (MARK)

Utilizando a tag (STYLE) no cabechalho, podemos denominar globalmente uma cor para todos os marks
